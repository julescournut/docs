---
title: cache:clear
---

## Description
Clear the cache

## Usage
```shell
cache:clear [options]
```

## Arguments
- `module`                    module to activate

## Options
- `--no-warmup`            Do not warm up the cache
- `--no-optional-warmers`  Skip optional cache warmers (faster)


## Example
```shell
php Thelia cache:clear
```
